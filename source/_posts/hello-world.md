这是我的第一篇博客文章！欢迎来到我的个人博客。

## 关于本站

本博客使用 [Hexo](https://hexo.io/) 框架搭建，主题为 [NexT](https://theme-next.js.org/)，托管在 GitHub Pages。

## 写作

```bash
# 创建新文章
npx hexo new "文章标题"

# 本地预览
npx hexo server

# 生成静态文件
npx hexo generate

# 部署到 GitHub Pages
npx hexo deploy
```

开始写作吧！
