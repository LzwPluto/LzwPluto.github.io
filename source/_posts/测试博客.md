# 博客测试文章：一站式体验常见博客排版元素

## 前言

这是一篇专门用于测试博客系统排版效果的综合性文章。它涵盖了博客写作中绝大多数常见的**块级元素**和**行内元素**，包括多级标题、代码块、表格、引用、列表、Tab 切换、图片占位、数学公式等。通过这篇文章，您可以快速检验博客主题的样式是否完整、响应式是否良好。

### 1.1 三级标题（H3）

这是三级标题下的正文内容。博客文章通常通过多级标题来组织内容结构，方便读者快速浏览和定位。

这是四级标题下的正文。某些博客主题支持 H4 及更深层级的标题样式，以提供更细致的结构化写作能力。

## 五、代码块测试

### 5.1 普通代码块（无语言）

```
function hello() {
  console.log("Hello, Blog Test!");
}
```

### 5.2 带语言标识的代码块（JavaScript）

```javascript
// JavaScript 示例：斐波那契数列
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

const result = fibonacci(10);
console.log(`Fibonacci(10) = ${result}`);
```

### 5.3 Python 代码块

```python
# Python 示例：列表推导式
squares = [x**2 for x in range(10) if x % 2 == 0]
print(f"偶数平方: {squares}")
```

本文档覆盖了博客写作中常见的绝大多数排版元素，包括：

- ✅ 多级标题（H2 ~ H6）

- ✅ 列表（无序、有序、任务列表、定义列表）
- ✅ 代码块（多种语言、无语言、带行号高亮）
- ✅ 表格（基础表格、对齐表格、复杂表格）
- ✅ 图片与媒体（普通图片、带尺寸图片）
- ✅ Tab 切换组件
- ✅ 数学公式（行内与块级）
- ✅ 分割线与特殊字符

如果以上所有元素在您的博客中均能正确渲染，说明博客主题的样式覆盖非常完整。如果有部分元素渲染异常，可以根据本文档定位具体问题并加以修复。
